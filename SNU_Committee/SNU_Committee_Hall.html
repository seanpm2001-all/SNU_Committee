Constitution V1.02 - December 6th 2019

Most recent version of constitution must be used under any circumstance until specified (it is in a working draft)

Distro rules

SNU distributions cannot use the committee system in their own distributions. It can only be used in the main distribution

SNU distributions cannot invade user privacy. Invasion of user privacy and data collection by distributions will result in repeated fines at the highest level of the law

SNU distributions cannot be used in a monopolistic fashion, you must make room for competition, and not prevent users from switching over to other distributions

SNU distributions cannot contain DRM

SNU distributions must be open source, except for closed source content stored on the site. With this, the public must be able to copy the source of the entire site when they want without requesting.

Management

SNU can be run in any country on any planet, no country can be denied access to owning a SNU distribution.

Privacy

Data collection on SNU distributions is prohibited. Collection of sensitive personal data is illegal, and will result in intense legal fines even to the point of bankruptcy.

Users must consent to all data collection requested without reading terms of service, or without it being flashed on them (the message has to appear for at least 180 seconds per viewing)

Certain data collection cannot be done even with consent. This certain data includes "medical records" "social security and other sensitive bank details"

Location data can be collected only when the user gives consent, and consent has to be given each time. Location data is important for features such as GPS data.

Malicious trackers and cryptominers are not allowed to be displayed on SNU.

Constitution updates

Constitution updates must be kept fair and simple. Principal rules cannot be revoked by anyone.
